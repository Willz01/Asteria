# Asteria
Theatre movies booking web project for Software Development for the Web, HKR.
----------------------------
### Setup backend (API)
> `$ cd backend`
> 
> `npm install`

### Run 'nodemon start script'
> `npm start` 
------------------------------
### Setup frontend (Web page)
> `$ cd frontend`
> 
> `npm install`

### Run 'nodemon start script'
> `npm start` 
>
> Index page served at http://127.0.0.1:5601/
